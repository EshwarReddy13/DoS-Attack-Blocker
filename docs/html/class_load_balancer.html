Detailed Description

Class that manages web servers and a queue of requests to simulate load balancing.

This class implements a load balancer that distributes incoming web requests across multiple web servers. It includes features for dynamic server scaling, request queue management, and performance monitoring. The load balancer can automatically add or remove servers based on current load to maintain optimal performance.

Additional features include IP range blocking for firewall/DOS attack prevention.
